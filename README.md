# UtilExample.java

## Overview
This project demonstrates how to build a simple Java utility class that handles user input and performs basic operations. It is designed as an introductory example for learning Java fundamentals such as:
- Console input/output
- Data processing
- Structured programming

## Features
- Reads input from the user
- Processes and validates data
- Displays results in a clear format
- Organized into a reusable utility class

## Tech Stack
- **Language:** Java
- **Environment:** JDK 17+ (or any modern Java version)
- **IDE:** Compatible with IntelliJ IDEA, Eclipse, or VS Code

## Getting Started
